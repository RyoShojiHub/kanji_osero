概要:
漢字オセロは、Pythonを使用したオセロゲームのアプリケーションです。通常のオセロと異なり、盤面のコマには漢字が使用されており、プレイヤーは漢字のコマを裏返しながらゲームを進めます。
就職活動で提出するポートフォリオの一環として制作しました。


使用方法:
プロジェクトのルートディレクトリに移動し、必要なパッケージをインストールします。
cd kanji_othello
pip install -r requirements.txt

ゲームを開始するには、以下のコマンドを実行してください。
python main.py

select_file.txtの1行目をkanji_dif.txtに書き換えることで、難易度を上げることができます。(漢検準1級相当)
